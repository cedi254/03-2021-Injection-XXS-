# OWASP A03:2021 – Injection (XSS)

This project demonstrates Cross-Site Scripting (XSS).

## Setup

Install dependencies:

npm install

Start the server:

npm start

Open in browser:

http://localhost:3000

## Demo

Unsafe version uses innerHTML.

Safe version uses textContent.

The goal is to demonstrate how unsafe rendering of user input can allow injection of unwanted HTML into a website.
